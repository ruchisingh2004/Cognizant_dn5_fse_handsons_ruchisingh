## Hands-On Labs

### HOL 1 - My First React Application

**Project Name:** `myfirstreact`

Created a basic React application and displayed a welcome message using a React component.

**Concepts Covered:**
- React environment setup
- Create React App
- React application structure
- JSX
- Rendering React components

---

### HOL 2 - Student Management Portal

**Project Name:** `studentapp`

Created a Student Management Portal containing Home, About, and Contact components.

**Concepts Covered:**
- React Components
- Class Components
- Multiple Components
- Component Rendering
- Import and Export of Components

---

### HOL 3 - Score Calculator Application

**Project Name:** `scorecalculatorapp`

Created a functional component to calculate and display the average score of a student using component properties.

**Concepts Covered:**
- Functional Components
- Props
- Component Styling
- CSS Stylesheets
- Rendering Components

---

### HOL 4 - Blog Application

**Project Name:** `blogapp`

Created a blog application that retrieves and displays posts using the Fetch API and React component lifecycle methods.

**Concepts Covered:**
- Class Components
- React State
- Component Lifecycle
- componentDidMount()
- componentDidCatch()
- Fetch API
- Error Handling

---

### HOL 5 - Cohort Tracker Application

**Project Name:** `cohortstracker`

Styled React components for a cohort tracking dashboard using CSS Modules and inline styling.

**Concepts Covered:**
- React Component Styling
- CSS Modules
- Inline Styles
- className
- Dynamic Styling

---

### HOL 9 - Cricket Application

**Project Name:** `cricketapp`

Created a cricket player application using ES6 features to display, filter, destructure, and merge player data.

**Concepts Covered:**
- ES6
- map()
- Arrow Functions
- Array Filtering
- Destructuring
- Spread Operator
- Array Merging

---

### HOL 10 - Ticket Booking Application

**Project Name:** `ticketbookingapp`

Created a ticket booking application that displays different pages for guest and logged-in users.

**Concepts Covered:**
- Conditional Rendering
- React State
- Login and Logout
- Event Handling
- Rendering Components Conditionally

---

### HOL 11 - Blogger Application

**Project Name:** `bloggerapp`

Created a blogger application containing Book Details, Blog Details, and Course Details components using conditional rendering.

**Concepts Covered:**
- Conditional Rendering
- Multiple Components
- List Rendering
- map()
- React Keys

---

### HOL 12 - Ticket Raising Application

**Project Name:** `ticketraisingapp`

Created a complaint registration application that accepts employee and complaint details and generates a reference number.

**Concepts Covered:**
- React Forms
- Controlled Components
- Form Submission
- handleSubmit()
- Textbox and Textarea Controls
- Event Handling

---

### HOL 13 - Mail Registration Application

**Project Name:** `mailregisterapp`

Created a registration form with validation for name, email, and password fields.

**Concepts Covered:**
- React Forms
- Form Validation
- Controlled Components
- Event Handling
- Form Submission
- Input Validation
